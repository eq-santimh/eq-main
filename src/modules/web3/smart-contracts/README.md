# Smart contracts module

Responsabilidades:
- Mantener artefactos (ABIs, metadata, typechain si aplica)
- Gestionar despliegues (deploy scripts, addresses por red)
- Exponer “adapters” para leer/escribir (interacción EVM, signing, etc.)

En este prototipo solo se crea el esqueleto de carpetas para crecer
sin mezclar Solidity/artefactos con la lógica del producto.
